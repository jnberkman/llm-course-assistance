What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Take this class!!! This was my favorite class of the semester. The workload is not very difficult, the grading is fair, and most of all, the class is fun. The readings are not long, but really make you think. Personally, I was nervous that this class focused on creative writing and performance as someone who is much more analytical and STEM focused. But, this class ended up being a fantastic space to push myself out of my comfort zone. Because section is so small, you will get to know the section leader and other students well, which makes it so much less nerve–wracking to practice speaking Spanish. Also, Xiomara is an absolutely wonderful section leader if you get her! She is so understanding, provides so much feedback, and her enthusiasm for the class made it impossible to not be excited about it too. :)
It is a very welcoming environment that allows you to speak freely and ask questions!
This is a very chill class (native speaker). Definitely give yourself time to read over the materials before class since they are a bit wordy or dense.
This is a really great course that you can have a lot of fun with. The assignments are all creative which makes them more fun. There isn't as much of a grammar/vocabulary component if that's what you're looking for. How the class is structured also depends on your TF.
TAKE THIS CLASS!!!!!!!!!!!!!!! At the beginning of this course I was an okay speaker, but this class had such an emphasis on speaking and performing, which genuinely helped my speaking and my accent (while still a bit American) improve so much. The reading/writing component of this class is a bit less intense, but you do read some wonderful plays and the projects/writing assignments are really not that much and engaging (though there are quite a few group projects). My class was really small and we all got quite close, which made coming to Spanish every Tuesday and Thursday very fun and chill. While I know this review area is about the *class*, if you can, get Xiomara as you teacher––she is INCREDIBLE and AMAZING!!!!!!!!!!!!! If you've taken Spanish 20/30 (like I did), note that this class is not the same. People here are taking Spanish 50 because they genuinely want to get better at the language and learn, not just because they have to for the language requirement. The vibes are SO much better, and I got much better at Spanish.
