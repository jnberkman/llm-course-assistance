Comments
You will learn a lot of Italian very fast if you take this course. It's the best and worst thing about it. Because of the fast pace you'll definitely need to put in some time outside of class familiarizing yourself with new concepts. But I had tons of fun doing that and in class, and it definitely pays off once you've got a strong base of knowledge in a new language! Professor Lieberman is hilarious and fun, if hard to read at times.
It can be very intimidating, but Italian is beautiful and extremely fun to learn
Take this class if you really want to learn! This class is difficult, but it is so rewarding. I was very eager to learn outside of class (watch movies, read short stories, listen to podcasts/music), and this eagerness really helps in the long run. Meeting five days a week is a lot, but I thought it was worth it. You learn so much by the end of the semester – you will be amazed! This is an assignment–heavy class, but I never felt so overwhelmed that I couldn't get things done. If you want to learn Italian and are up for putting in the work, take this class! I was spending probably 10 hours per week on this course (during a good week).
I would say it is challenging but definitely worth it if you would like to be fluent and are passionate about language study.
